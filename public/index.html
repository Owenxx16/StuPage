STU WEBSITE CLONE Demo & Preview

A replica of the Saigon Technology University website built with modern web technologies

Project Overview

This project is a clone of the Saigon Technology University (STU) website, built using Next.js, React, and TailwindCSS. Below are screenshots of various sections we've implemented.

Technologies Used

Next.js 15
React 18
TailwindCSS
ShadCN UI
TypeScript

Header & Navigation

The header includes the school logo, language switcher, search functionality, and a comprehensive dropdown navigation menu.

Hero Carousel

Interactive carousel featuring information about admissions and school events, matching the original STU design.

News Sections

News and updates from the university presented in a clean, card-based layout with tabs for different categories.

Footer

Comprehensive footer with links to departments, admissions information, and contact details.
